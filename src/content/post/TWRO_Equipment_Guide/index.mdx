## Starter

Budget-friendly build, a bare minimum to progress to level 260. Keep in mind that there might be unorthodox low budget build for certain classes such as Rune Knight Peuz Set,
which I will create a separate post for it.

The best class with this build is Inquisitor TFB with the following build:
1. Acolyte: Auto-Attack
2. Monk/Champion: Raging Thrust
3. Sura: Sky Blow -> Tiger Cannon (TC)
4. Inquisitor: Third Flame Bomb (TFB)
