## Starter

Budget-friendly build, I've been using this build mainly, because I want to save a lot of Zeny to buy EP19 gears. Though, I don't advise magic classes at the start due to a higher cost.

The best class with this build is Inquisitor TFB with the following skills:
1. Acolyte: Auto-Attack
2. Monk/Champion: Raging Thrust
3. Sura: Sky Blow -> Tiger Cannon
4. Inquisitor: Third Flame Bomb (TFB)
