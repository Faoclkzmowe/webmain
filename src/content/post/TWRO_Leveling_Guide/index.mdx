## Preparation

### DAIDAI ★

![DAIDAI](public/posts/daidai.png)
DAIDAI is an NPC that offers a daily quest which gives EXP buff by 100% for 1 hour.
The quest can be accepted since level 1.

<Collapse title="Quest">
<span>Accepting: select 2nd option, which will give you a quest to kill specified monster.<br></br>
Turning in: select 2nd option, then 2nd option again.<br></br>
Giving up: select 3rd option, then 2nd option, but you won't get the buff that day.<br></br>
Reset time: 4AM UTC+8</span>
</Collapse>

### Town Mules

Town Mules are easy to make and always useful.

<Collapse title="Merchant">
<span>The first mule that should be created for increasing Zeny efficiency due to
[Overcharge](https://www.divine-pride.net/database/skill/38), [Discount](https://www.divine-pride.net/database/skill/37), and preferably
[Vending](https://www.divine-pride.net/database/skill/41) because you will run a lot of instances in the future, which means selling stuff to players.</span>
</Collapse>

<Collapse title="Acolyte (Warp Portal)">
<span>Save travel time by using [Warp Portal](https://www.divine-pride.net/database/skill/27)</span>
</Collapse>

### Battle Mules

Battle Mules are support classes that will help your main character by boosting damage, movement speed, hp, sp and etc.
Best use for solo leveling and leveling quests.

<Collapse title="Cardinal">
<span>Obviously the best support class as it's meant to be a support class.</span>
</Collapse>

<Collapse title="Troubadour/Trouvere">
<span>Solo buff is possible thanks to [Kvasir Sonata](https://www.divine-pride.net/database/skill/5352).<br></br>
Troubadour is better than Trouvere for the most case because of ACDR skill aka [Magic Strings](https://www.divine-pride.net/database/skill/321).
They can also be used for running hunting quests because of one of the best AoE skill aka [Metallic Fury](https://www.divine-pride.net/database/skill/5356) (260/55 patch onwards only, otherwise trash skill).</span>
</Collapse>

### 3rd Job (100-200)

As soon as you hit level 108, you should always go for party leveling as it's like 3-4x faster, no more Multi-Client, a lot of buffs, and leader will always have a Warp Portal too.
So please, just party leveling. Don't forget to do Eden Group Leveling Quests to rent 3 days Advanced Paradise Shadow Equipment for 20 Eden Coin.

## Leveling Quests

<div>
  <Collapse title="Pros">
    <ul style="padding: 0; margin: 0; list-style-position: inside;">
      <li style="padding: 0; margin: 0; list-style-position: inside;">Better than Party Leveling, if running multiple characters at once.</li>
      <li style="padding: 0; margin: 0; list-style-position: inside;">Obtain Eden Coin which can be used for renting Eden Shadow Equipment and buying Eden Gears.</li>
      <li style="padding: 0; margin: 0; list-style-position: inside;">No need to invest in equipment until you want to continue Party Leveling, 210 is a great spot since you can wear EP19 gears.</li>
    </ul>
  </Collapse>
</div>

<div>
  <Collapse title="Neutral">
    <ul style="padding: 0; margin: 0; list-style-position: inside;">
      <li style="padding: 0; margin: 0; list-style-position: inside;">Worth doing from level 150 to 210.</li>
    </ul>
  </Collapse>
</div>

<div>
  <Collapse title="Cons">
    <ul style="padding: 0; margin: 0; list-style-position: inside;">
      <li style="padding: 0; margin: 0; list-style-position: inside;">Worse than Party Leveling, if solo.</li>
      <li style="padding: 0; margin: 0; list-style-position: inside;">Required high level Troubadour/Trouvere and Arch Bishop/Cardinal.</li>
    </ul>
  </Collapse>
</div>

<div>
  <Collapse title="Requirements">
    <ul style="padding: 0; margin: 0; list-style-position: inside;">
      <li style="padding: 0; margin: 0; list-style-position: inside;">Carry class such as Troubadour/Trouvere.</li>
      <li style="padding: 0; margin: 0; list-style-position: inside;">Arch Bishop for faster progression, as you will teleport until you find a large group of monsters, then use Convenio.</li>
    </ul>
  </Collapse>
</div>
